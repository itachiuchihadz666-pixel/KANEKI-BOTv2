# KANEKI-BOT v2 🤖

بوت فيسبوك ماسنجر مع 4 قوائم أوامر تفاعلية وتحديث تلقائي للكوكيز.

## التشغيل

1. أضف ملف `appstate.json` يحتوي على كوكيز فيسبوك
2. نفّذ `npm install`
3. شغّل البوت بـ `npm start`

## الاستخدام

- أرسل `الاوامر 1` → يصلك GIF
- رد على الـ GIF → تظهر لك قائمة الأوامر الكاملة
- نفس الشيء مع `الاوامر 2` و `الاوامر 3` و `الاوامر 4`

## الميزات

- ✅ 4 قوائم أوامر بـ GIF تفاعلي
- ✅ تحديث تلقائي للكوكيز كل ساعة
- ✅ Railway deployment جاهز
- ✅ إعادة تشغيل تلقائية عند الفشل

## Railway

جاهز للنشر على Railway مباشرة. أضف `appstate.json` كـ environment variable أو volume.
